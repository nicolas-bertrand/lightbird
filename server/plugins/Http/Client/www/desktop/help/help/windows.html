Les fenêtres.

Les fenêtres du client web sont similaires à celle que vous utilisez dans votre système d'exploitablion. Vous pouvez les bouger, les redimensionner, les cacher, et les maximiser.

Les quatres bouttons à gauche et à droite du titre d'une fenêtre permettent de la gérer :

- Le boutton rouge permet de fermer une fenêtre.
- Le jaune la cache. Elle sera toujour ouverte, et pourra être affichée à n'importe quel moment.
- Le lien vert agrandi la fenêtre de toute la taille de l'écran. Cliquer à nouveau dessus remet la fenêtre à sa taille d'origine.
- Enfin, le boutton bleu permet de cacher toutes les autres fenêtres. Cliquer dessus lorsque les autres fenêtres sont cachés les réaffichera.
